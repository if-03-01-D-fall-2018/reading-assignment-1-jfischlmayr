Reading Assignment 1: WWW and html

World Wide Web General

  1. What are the most essential technologies that made the World Wide Web such a success?

    • Mosaic's graphical user interface
    • The universal document identifier (UDI), later known as uniform resource locator (URL) and uniform resource identifier (URI)
    • The publishing language HyperText Markup Language (HTML)
    • The Hypertext Transfer Protocol (HTTP)

  2. How are resources (html pages, images, etc) addressed in the web?

    • With URL's. They assign names like "www.example.com" to IP adresses.

  3. What are the general components of a URL?

    • A non-empty scheme component followed by a colon (:) like: http, https, ftp, mailto, file, data, and irc
    • An optional non-empty authority component preceded by two slashes (//)
      • An optional userinfo subcomponent
      • A non-empty host subcomponent
      • An optional port subcomponent preceded by a colon (:)
    • A path component, consisting of a sequence of path segments separated by a slash (/)
    • An optional query component preceded by a question mark (?)
    • An optional fragment component preceded by a hash (#)
